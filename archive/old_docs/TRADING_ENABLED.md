## ⚙️ 如果需要暫停交易

### 方法 1: 透過 Railway 控制台

1. 訪問：https://railway.com/dashboard
2. 選擇項目：ravishing-luck → 服務：winiswin
3. Variables 標籤
4. 修改：`ENABLE_TRADING=false`
5. 保存（機器人會自動重啟並停止交易）

### 方法 2: 使用命令行

```bash
# 在 Replit Shell 中執行
railway variables --set "ENABLE_TRADING=false" --service winiswin
```
